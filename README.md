# AICRA v2 – AI Powered Code Review Assistant

An AI-powered web application that helps developers analyze and improve their code through intelligent reviews and feedback. Users can submit code snippets to receive AI-generated suggestions, identify potential issues, and learn coding best practices in a simple and interactive interface.

##  Features
- AI-powered code analysis
- Code quality improvement suggestions
- Error detection and feedback
- Clean and responsive user interface
- Database integration for managing data

##  Tech Stack
- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQL Server

##  Purpose
This project was developed to simplify the code review process by providing instant AI-generated feedback. It helps developers or even students identify mistakes, improve code readability, and follow better programming practices.
