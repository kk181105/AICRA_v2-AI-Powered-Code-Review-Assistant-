# Backend

This folder contains the backend implementation of the AI Code Review & Learning Assistant, developed using Python and Flask. It manages the application's core functionality, including request handling, business logic, database connectivity, AI-powered code review, and the Game Arena module. The folder also includes the `templates` directory, which stores the HTML files used to render the application's web interface.

## Contents
- Python source files for backend logic - app, db(database collector), runner, ai_agent
- Flask application and routing
- Database integration
- AI code review functionality
- Game Arena backend module
- `templates/` folder containing HTML pages
