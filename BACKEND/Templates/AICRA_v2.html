Project Problem Statement
AI-Powered Code Review & Learning Assistant for Python
Python developers frequently commit fragile, insecure, or poorly-typed code due to insufficient automated feedback and no structured way to learn from their mistakes. AICRA addresses this by providing real-time AI code review using three intelligent agents, multi-line contextual corrections stored in a Python-backed SQLite database, and a gamified learning module that adapts challenges dynamically based on detected error patterns. This ensures developers don't just fix bugs — they understand and remember why.
