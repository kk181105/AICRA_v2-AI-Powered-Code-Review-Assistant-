
# Frontend

This folder contains the frontend of the AI Code Review & Learning Assistant. It includes the HTML pages that provide the user interface for the application, allowing users to navigate through features such as code submission, AI-generated review results, authentication, and the Game Arena module. The frontend is designed to deliver a clean, responsive, and user-friendly experience while seamlessly interacting with the Flask backend.

## Contents
- HTML pages for the website
- User interface for code review
- Login and authentication pages
- Dashboard and navigation pages
- Game Arena interface
